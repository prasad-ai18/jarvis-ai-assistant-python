# Jarvis AI Assistant

This is a simple AI voice assistant built using Python.

## Features
- Voice command recognition
- Search information from Wikipedia
- Tell the current time
- Open Google using voice command
- Exit assistant using voice command

## Technologies Used
- Python
- SpeechRecognition
- pyttsx3
- Wikipedia

## How to Run

Install required libraries:

pip install pyttsx3 speechrecognition wikipedia

Run the program:

python main.py
